Ventilation

  • Ventilation system performance is integral to the comfort and safety of the occupants.
  • Poor ventilation can lead to human response symptoms of sick building syndrome: Headache, shortness of breath, mental fatigue, and absenteeism.
  • Inadequate ventilation major contributor to the prevention of aerosol transmission of respiratory viruses.



  1. Measured ventilation rate.
    • Ventilation rate was calculated using the observed decay of the natural tracer gas carbon dioxide concentration during times of reducing occupancy.
    • Scoring: Average ventilation rate for each space is scored based on thresholds derived from ASHRAE 62.1-2010 and WELL v2 A06 Enhanced Ventilation.
  2. Short-term carbon dioxide exposure.
    • Effectiveness of the ventilation system to quickly remove or disperse large sources of aerosols.
    • Scoring: Percentage time carbon dioxide short-term averages levels spent in each threshold range derived from literature.
  3. Negative pressure analysis.
    • Reduces the risk of microorganisms (Coronavirus) being spread by airborne transmission and odor spread.
    • Scoring: Percentage time spent in each threshold range ASHRAE Standard 170.

Air Exchange Rate Algorithm

The air exchange rate assessment was based on the methodology outlined by Batterman [1]. Batterman reviewed the method of using carbon dioxide as a natural tracer gas to determine ventilation rates specifically for the application to school classrooms. The general method is based on the analysis of a single well-mixed space as shown in Figure 1.


By applying a mass balance on the control volume in Figure 1 the mass balance model is generated.

Decay Method

During periods of stepwise decrease in occupancy, exponential decay of indoor CO_2 concentration will be observed by sensors. The exponential decay was utilized to determine the air exchange rate in the space. The decay air exchange rate, A_D, for a single fully mixed zone can be determined using a sequence of CO_2 concentrations, C_t. During ioAirFlow tests, the CO_2 concentrations of the space, C_t, and the CO_2 concentrations of the supply air, C_R, are measured while the steady state concentration C_s is not explicitly known. With the decay method, this lack of information is acceptable since the concentration above the return air is being plotted with respect to time and the slope of the line is the only important parameter. The estimated steady state could be calculated based on the intercept of the lines.


Decay Air Exchange Rate Algorithm 

Initialize Start at beginning of natural log CO_2 concentration signal (ind_start).
Initialize Set end index from delay value (ind_end).
Initialize Linear regression score R^2 as 1 
While there is more data in the signal to analyze do
	Initialize start-up phase 
	Initialize line has not been found
	Fit the data sliced at the index start and end [ind_start: ind_end]
	If the last value is equal to zero, a new line should start since a new line will always be made at zero.
	If the linear regression score R^2 is greater than the previous step stop startup phase.
	If startup phase is over do
		If regression score  R^2 is less than the previous step, break the linear line at the previous step, associated with the greatest regression score. 
		Return Line parameters and score.
	If startup is not over do
		Update ending index to include next point, current regression score as previous score. 
	If the end index is greater than the size of the  signal do
		break both loops

Build Up Method

Conversely, the air exchange rate can be determined using periods of increased CO_2 concentrations from a step-wise increase in occupancy. The build-up air exchange rate, A_B, for a single fully mixed zone can be determined using a sequence of CO_2 concentrations, C_t.


BuildUp Air Exchange Rate Algorithm 

Find Indices of inflection points from concave up to down. 
Set Start indices as inflection points and end indices as the next sequential inflection point.
For each pair of start and end indices do
	Fit exponential function to sliced data
	Return function parameters
Select the best parameters to be used as air exchange rate estimation.
